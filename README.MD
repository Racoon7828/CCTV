# 서울시 치안 데이터 대시보드

서울시 25개 자치구의 CCTV 설치 현황, 범죄 발생 이력, 인구 통계를 통합 분석하여
치안 인프라의 효율성과 CCTV 투자 우선순위를 시각화하는 Streamlit 대시보드입니다.

- **분석 기간**: 2015 ~ 2024년
- **분석 대상**: 서울시 25개 자치구
- **분석 방법**: 피어슨 상관계수, 순위 종합 분석

---

## 분석 내용

### CH1. 지역구별 분석 (2024년 기준)

2024년 기준 자치구별 주요 변수 간 상관관계를 분석합니다.  
각 섹션은 **추이 분석(막대+꺾은선)** 과 **산점도(회귀선 포함)** 두 가지 형태로 전환하여 볼 수 있습니다.

| 섹션 | 내용 |
|------|------|
| 1. CCTV 수 & 범죄 발생 건수 | CCTV 설치 수 오름차순 정렬 후 범죄 발생 건수와 비교. 피어슨 상관계수(r) 제공 |
| 2. 유흥업소 수 & 범죄 발생 건수 | 영업 중인 유흥주점 수와 범죄 발생 건수 간 상관관계 분석 |
| 3. 인구 수 & 범죄 발생 건수 | 등록인구 오름차순 정렬 후 범죄 발생 건수와 비교. 자치구별 등록인구 현황 바 차트 포함 |
| 4. CCTV 수 & 인구 대비 범죄율 | 단순 설치 대수가 아닌 인구 10만명당 범죄율과 비교. 상관관계 미약함을 시각적으로 확인 |
| 종합 데이터 요약 | 자치구별 인구수·CCTV수·범죄발생건수·범죄율·CCTV밀도·CCTV당범죄건수 통합 표 |

> 사용 지표: 범죄율 = 발생건수 / 인구수 × 100,000 / CCTV 밀도 = CCTV수 / 인구수 × 10,000 / CCTV당 범죄수 = 발생건수 / CCTV수

### CH2. 범죄 유형별 분석

**1. 범죄 유형별 발생 추이 및 CCTV 상관성 (2015–2024)**

살인·강도·절도·폭력·강간강제추행 5가지 유형을 버튼으로 선택(중복 선택 가능)하여 분석합니다.

- **Plotly 인터랙티브 차트**: CCTV 누적 설치량(막대)을 배경으로 선택한 범죄 유형별 발생 건수(꺾은선) 오버레이
- **연도별 증감 테이블**: 선택 범죄 유형의 전년 대비 증감 수치를 ▲▼로 표시
- **CCTV–범죄 상관계수**: 선택 범죄 합계와 CCTV 누적량의 피어슨 r 값 자동 계산
- **2015→2024 변화율 요약**: CCTV 증가율과 각 범죄 유형의 증감률(%) 메트릭 카드

**2. 5대 범죄 발생 장소 분포 (2024년)**

12개 장소 유형(아파트/연립다세대, 단독주택, 도로/골목길, 음식점/유흥업소 등)별 범죄 발생 비율을 분석합니다.

- **히트맵**: 범죄 유형 × 발생 장소 비율(%) 전체 개요
- **탭별 상세 차트**: 범죄 유형별로 발생 장소 비율 수평 막대 차트 제공

### CH3. 연도별 추세 분석 (2015–2024)

**1. 서울시 CCTV 연도별 설치 현황**

3개 탭으로 구성됩니다.

| 탭 | 내용 |
|----|------|
| 연도별 누적 설치 총량 | 25개 자치구 합산 연도별 누적 CCTV 설치 대수 막대 차트 |
| 연도별 신규 설치 추이 | 전년 대비 신규 설치 대수 꺾은선+면적 차트 |
| 자치구별 2024년 CCTV | 자치구별 누적 대수 수평 막대 차트 (평균 기준 색상 구분) |

**2. CCTV 증가량 & 범죄율 감소 상관관계**

- CCTV 누적 수량(좌축)과 5대 범죄율 인구 10만명당(우축)을 이중 축 차트로 비교
- CCTV 증가율, 범죄율 변화율, CCTV–범죄율 피어슨 상관계수(r) 메트릭 카드 제공

**3. CCTV 인프라 & 실시간 탐지 신고 건수 상관관계 (2021–2024)**

- CCTV 누적 설치 수(막대)와 CCTV 실시간 탐지 실적(꺾은선)을 Plotly 인터랙티브 차트로 비교
- 인프라 확충 대비 탐지 효율 상관계수(r) 제공

### CH4. CCTV 투자 우선순위

자치구별 CCTV 추가 설치 우선순위를 세 가지 지표를 종합하여 도출합니다.

**부담 지표: CCTV 1대당 범죄 건수**
- CCTV 1대가 감당하는 범죄 건수를 자치구별로 비교
- 수치가 높을수록 CCTV가 상대적으로 부족한 자치구

**종합 우선순위 산출 방식**

```
종합점수 = (범죄율 순위 + CCTV밀도 역순위 + CCTV당범죄수 순위) / 75 × 100
```

- **범죄율 순위**: 인구 10만명당 범죄율이 높을수록 높은 순위
- **CCTV 밀도 역순위**: 인구 1만명당 CCTV 수가 적을수록 높은 순위
- **효율성 순위**: CCTV 1대당 범죄 건수가 많을수록 높은 순위

결과는 종합점수 기준 전체 자치구 순위표(히트맵 컬러 적용)와 **TOP 10 우선순위 수평 막대 차트**로 제공됩니다.
